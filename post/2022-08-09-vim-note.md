# Vim Note

## vim basic command

*Vim的几种模式*
- 正常模式：可以使用快捷键命令，或按`:`输入命令行
- 插入模式：可以输入文本，在正常模式下，按`i`、`a`、`o`等都可以进入插入模式。
- 可视模式：正常模式下按`v`可以进入可视模式，在可视模式下，移动光标可以选择文本。
按`V`进入可视模式，总是整行整行的选中。`ctrl+v`进入可视块模式。
- 替换模式：正常模式下，按`R`进入。
